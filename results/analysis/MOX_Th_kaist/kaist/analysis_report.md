## 1. 概述

### 1.1 算例背景
KAIST MOX-Th 基准题是用于研究**钍基压水堆可燃毒物控制**的中子学基准。18 个有效算例按两个维度展开（命名约定来自 `DATA_DICTIONARY.md`：`M{pu_pct*10}-{th_pct}`，例 `M43-30` = 4.3% Pu + 30% Th）：

| 维度 | 取值 |
|---|---|
| **MOX 钚含量** | M43 (4.3% Pu)、M70 (7.0% Pu)、M87 (8.7% Pu) |
| **Th 含量** | 5、10、15、20、25、30 wt%（数字越大 Th 越多） |

每个算例定义三个 `th232_capN` 探测器，`dm` 标签分别指向三类燃料，**本分析只取与本算例 `dm` 匹配的探测器**（如 M87-20 → `th232_cap3`），避免把其他族的燃料贡献算进来。

### 1.2 物理目标
1. Th-232(n,γ) 微观反应率的能谱形状（70 群，per lethargy）
2. 配 BOC 原子数密度后得到宏观反应率（per cm³ per lethargy）
3. 厘清**微观率排序**与**宏观率排序反转**的物理机制
4. 评估**共振自屏蔽**在不同能群的表现
5. 摸清 MOX 配比与 Th 含量对俘获谱形的耦合影响

## 2. 数据流（从原始 SERPENT 到出图）

```
SERPENT 输入 (.m input file)
    │
    ├─> _dep.m      ←─── MAT_<case>_ADENS (Th232 原子数密度 per burnup step)
    │                 BU 全局燃耗轴 (26 个时刻)
    │
    ├─> _res.m      ←─── ANA_KEFF, CONVERSION_RATIO
    │
    └─> _det0.m     ←─── DETth232_capN (70 群微观率 per lethargy)
                      DETth232_capNE (能群边界)
                      N 由输入卡 `det th232_capN n dm <case>` 匹配
```

| 步骤 | 脚本 | 输出 |
|---|---|---|
| 1. 解析输入卡，定位 capN | `find_th232_detector_suffix` | N ∈ {1,2,3} |
| 2. 读 70 群微观率 | `read_capture_sum` | (E_mid, micro_R) |
| 3. 读 BOC 原子数密度 | `read_th232_adens` | float |
| 4. 计算宏观率 | `read_macroscopic_capture` | (E_mid, micro_R × N) |
| 5. 出图 | `plot_response_by_family`, `plot_normalized` | PNG + CSV |

## 4. 物理分析

### 4.1 Th-232 俘获谱形：21.8 eV 巨共振主导

70 群能谱在所有 18 个算例中都呈现**单峰结构**：
- **峰位**：能群 30，E_mid = 2.225×10⁻⁵ MeV ≈ **21.8 eV**（Th-232 巨共振）
- **次级峰**：能群 36 附近，约 200 eV（次级共振）
- **热端**（E < 5×10⁻⁷ MeV，1/v 区）：缓升
- **快端**（E > 10⁻¹ MeV）：3-6%，随 MOX 配比增加而升高

| 区域 | 能量范围 | 占比范围 |
|---|---|---|
| 热区 Thermal | E < 5×10⁻⁷ MeV | 4.5–11.6% |
| 超热区 Epithermal | 5×10⁻⁷ ≤ E < 10⁻⁴ MeV | 36.1–51.9% |
| 共振区 Resonance | 10⁻⁴ ≤ E < 10⁻¹ MeV | 38.6–49.1% |
| 快区 Fast | E ≥ 10⁻¹ MeV | 3.3–6.2% |

**趋势**：随 Th 含量增加，热区份额↑（Th 多→谱软化），快区份额↑（共振吸收↑→更多快中子泄漏到共振区上方）。随 MOX 配比增加（M43→M70→M87），共振区份额↑（谱硬化），热区份额↓。

### 4.2 排序反转：微观率 vs 宏观率

**核心现象**（以 M87 族为例）：

| 算例 | N_Th232 | 微观总量 | 宏观总量 |
|---|---|---|---|
| M87-5 | 1.211e-3 | **1.990e+16** ← 最高 | **2.411e+13** ← 最低 |
| M87-30 | 7.267e-3 | 1.132e+16 ← 最低 | 8.224e+13 ← 最高 |

排序完全反转。**算术分解**：

| 因子 | M87-30 / M87-5 | 物理含义 |
|---|---|---|
| 密度比 N(M87-30) / N(M87-5) | **6.00×** | M87-30 多了 6 倍 Th 原子 |
| 微观率比 micro(M87-5) / micro(M87-30) | **1.76×** | M87-5 每个原子看见更多通量 |
| 宏观率比 macro(M87-30) / macro(M87-5) | 3.41× | = 6.00 / 1.76 |

**结论**：6.0× 密度涨幅 > 1.76× 自屏蔽跌幅，所以宏观率反转。**反转的物理本质**是：微观率反映"每个 Th 原子被照到的程度"，宏观率反映"每 cm³ 里发生多少反应"。前者是密度无关的"被照度"，后者必须把原子总数乘进去。

### 4.3 共振自屏蔽：每群量化

取 M87-5 vs M87-30（同一族、Th 含量极差最大），逐群比 micro(5) / micro(30)：

| E_mid (MeV) | 区域 | ratio (M87-5 / M87-30) | 物理 |
|---|---|---|---|
| 1.20e-9 | 1/v 热端 | 0.97 | 无自屏蔽 |
| 5.74e-6 | 共振谷 | 0.97 | 无自屏蔽 |
| **2.23e-5** | **21.8 eV 巨共振** | **2.73** | **强自屏蔽** |
| 1.70e-4 | 次级共振 | 1.90 | 强自屏蔽 |
| 9.25e-4 | 共振后 | 1.17 | 中等 |
| 2.74e-2 | 快区 | 0.99 | 无自屏蔽 |
| 1.71e+1 | 14 MeV | 1.00 | 无自屏蔽 |

**自屏蔽效应只在共振能群出现**，远离共振（热端、1/v、快端）每个原子看到的反应率几乎相同。这与**共振吸收 + 通量塌陷**理论完全一致。

**机制**：
- 21.8 eV 共振吸收截面高达 ~7000 barn
- 当 N_Th232 高时，**燃料块表面**的 Th 原子就吃光了 21.8 eV 附近的中子
- 内部 Th 原子"看不见"共振中子 → 微观率（per atom）下降
- 这种下降按 **1/√N 趋势**（NR 近似）粗略估算

### 4.4 MOX 钚含量影响

同 Th 含量、不同 Pu 含量（取 -5 系列）：

| 族 | Pu 含量 | 微观总量 | 谱形（Th/Ep/Re/Fa %） |
|---|---|---|---|
| M43-5 | 4.3% | 2.09e+16 | 6.82 / 51.36 / 38.57 / 3.25 |
| **M70-5** | 7.0% | **3.89e+16** | 5.60 / 51.83 / 39.20 / 3.37 |
| M87-5 | 8.7% | 1.99e+16 | 4.51 / 51.89 / 40.08 / 3.52 |

**为什么 M70 最高？**

- **M43（Pu 少，4.3%）**：Pu-239/241 太少 → 中子产额不足 → 总通量水平低
- **M70（Pu 中，7.0%）**：中子产额充足 + 慢化谱合理 → 通量水平+谱形都最优
- **M87（Pu 多，8.7%）**：中子产额最高，但谱太硬 → 大量快中子没被 Th 慢化就被吸收 → per-atom 率反而低

**耦合效应**：
- Th 含量从 5% 增加到 30% 时，**微观率下降斜率**：M70（3.89→2.25，降 42%）< M43（2.09→1.21，降 42%）≈ M87（1.99→1.13，降 43%）
- 即 Pu 含量主要影响"截距"，不显著影响"斜率"
- 斜率主要来自自屏蔽（~1/√N 趋势）

### 4.5 自屏蔽分类与本文覆盖度

| 自屏蔽类型 | 物理 | 当前数据能否看到 | 后续工作 |
|---|---|---|---|
| **能量自屏蔽** | 共振能群通量塌陷 | ✅ 已量化（21.8 eV 2.73×） | 拟合 NR 系数 |
| **空间自屏蔽** | 燃料棒内径向梯度 | ❌ `dm <case>` 是体平均 | pin-by-pin tally |
| **Doppler 展宽** | 温度上升→共振加宽→峰下降 | ❌ 当前单温度 900 K | 多温度扫描 |
| **互屏蔽** | U-238 / Pu-239 共振影响 Th 谱 | ❌ 需分解多核素 | 多探测器同步分析 |

## 5. 物理机制综合图

```
Th-232(n,γ) 微观反应率 (per atom)
    │
    ├─ σ_γ(E)        ← ENDF/B-VII.1 截面（21.8 eV ~7000 b）
    ├─ φ(E)          ← 中子通量
    └─ Δu            ← lethargy 宽度

微观率 = ∫ σ_γ(E) × φ(E) d(ln E)    per source neutron

Th-232 原子数密度 N (atoms/barn/cm)
    ↓
宏观率 = N × 微观率    per cm³ per source
```

| 量 | 谁影响 | 当前数据范围 |
|---|---|---|
| σ_γ(E) | 核数据 | 固定 ENDF/B-VII.1 |
| φ(E) | MOX 配比 + Th 含量 + 几何 + 慢化 | 18 算例扫描 |
| Δu | 70 群结构 | 固定 |
| N | Th 含量 | 1.2e-3 – 7.3e-3 |
| 总几何 | KAIST 17×17 栅格 | 固定 |

**只有 φ 和 N 在变**。所以看到的现象都来自这两个量的耦合。

## 6. 关键定量结论

1. **峰值位置**：2.225×10⁻⁵ MeV = 21.8 eV（Th-232 巨共振），与核数据一致。
2. **微观率峰值族**：M70 > M43 > M87（M70 族的每个 Th 原子被照得最厉害）。
3. **微观率单调下降**：固定族时，Th 含量↑ → per-atom 率↓（自屏蔽）。
4. **宏观率排序反转**：宏观率随 Th 含量单调↑，因为 6× 密度涨幅 > 1.76× 自屏蔽跌幅。
5. **共振自屏蔽只在共振能群**：21.8 eV 比 2.73×，200 eV 比 1.90×，热/快区比 ≈ 1.0。
6. **能区份额范围**：热 4.5-11.6%，超热 36-52%，共振 38-49%，快 3-6%。
7. **最佳 Pu 含量**：M70（7.0% Pu）在三个族中每原子俘获率最高，是 per-atom 俘获效率最优的设计点。
